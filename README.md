# EdgeAI-Based Online Exam Proctoring System

## Project Overview

This website was developed for the final year research project titled **“EdgeAI-Based Online Exam Proctoring System”**.

The project focuses on developing a lightweight, real-time, privacy-preserving online exam monitoring system using Edge AI. The system is designed to detect suspicious exam activities locally on the student’s device without depending on cloud-based processing.

## Main Project Components

1. **Object Violation Detection**  
   Detects prohibited objects such as mobile phones, books, notes, and unauthorized persons using webcam-based object detection.

2. **Behavioral Anomaly Tracking**  
   Tracks suspicious student behaviors such as gaze changes, head movements, and unusual motion patterns.

3. **Audio Integrity Verification**  
   Detects background voices, second-person speech, and audio-based exam violations.

4. **Continuous Biometric Authentication**  
   Verifies student identity continuously using facial biometric features.
