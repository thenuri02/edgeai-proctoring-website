Lightweight • Real-Time • Privacy-Preserving

EdgeAI-Based Online Exam Proctoring System

A scalable online exam monitoring system that detects suspicious activities in real time using lightweight AI models running locally on the student’s device.

Explore Project

Core Focus

Real-time fraud detection without cloud dependency.

Domain

This research focuses on building a lightweight Edge-AI proctoring framework for secure, private, and low-latency online examinations.

Literature Survey

Existing online proctoring systems mostly depend on cloud-based processing, creating concerns related to latency, bandwidth, scalability, and privacy.

Research Gap

Many current systems lack real-time local inference and still transmit sensitive audio, video, and biometric data to external servers.

Research Problem

How can online exams be monitored securely and accurately in real time while reducing cloud dependency and protecting student privacy?

Research Objectives

To develop a privacy-preserving Edge-AI proctoring system that detects cheating behaviors and verifies student identity locally.

Methodology

The system integrates object detection, behavioral tracking, audio integrity verification, and biometric authentication using optimized AI models.

Technologies Used

Python, TensorFlow Lite, TinyML, YOLO, MobileNet, FaceNet, CNN, RNN, OpenCV, and local dashboard technologies.
