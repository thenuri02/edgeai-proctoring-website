Lightweight • Real-Time • Privacy-Preserving

EdgeAI-Based Online Exam Proctoring System

This research presents a scalable and privacy-preserving online exam monitoring system that detects suspicious activities in real time using lightweight AI models executed locally on student devices.

Explore Project

Core Focus

Real-time fraud detection without cloud dependency.

Domain

This research focuses on designing a lightweight Edge-AI framework to ensure secure, private, and low-latency online examinations.

Literature Survey

Existing proctoring systems rely heavily on cloud-based processing, leading to increased latency, bandwidth usage, and privacy concerns.

Research Gap

Current systems lack efficient real-time local processing and still transmit sensitive biometric and behavioral data externally.

Research Problem

How can online exams be monitored securely and accurately in real time while minimizing cloud dependency and protecting user privacy?

Research Objectives

To develop a privacy-preserving Edge-AI system capable of detecting cheating behaviors and verifying identity using on-device intelligence.

Methodology

The system integrates object detection, behavioral tracking, audio verification, and biometric authentication using optimized TinyML-based models.

Technologies Used

Python, TensorFlow Lite, TinyML, YOLO, MobileNet, FaceNet, CNN, RNN, OpenCV, and lightweight on-device deployment frameworks.
